系统介绍

    该系统总共有三种登录角色：管理员，老师，学生

    管理员：　　　管理前台的显示内容，管理不同角色的不同权限，创建学生和老师的信息等等

    老师：　　　　　公布作业，批改作业，平分学生作业等等

    学生：　　　　　查看老师的作业公告，下载题目，上传作业等等
